# kotlin-study

코틀린으로 웹의 전반적인 기능을 구현합니다. 

메인 브랜치는 최신 버전의 기능을 담고 있습니다.

각 기능 버전별로 브랜치를 가집니다.


### 기능 정의(v1)
- 로그인/회원가입 기능
  - 회원가입(닉네임, 이메일, 패스워드) + 이메일 인증포함(시간내에 인증번호 작성)
- 게시판(다수의 첨부파일 포함 + aws s3)
  - 제목, 내용, 작성자, 첨부파일


### 기능 정의(v2)
- OAuth2 기능 추가(kakao, naver, google)
